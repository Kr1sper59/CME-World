# Locator bar

**Locator bar** - Кастомный плагин сервера CME World, созданный для управления точки, показывающей игрока на ванильном локатор-баре, добавленом в версии 1.21.8.

---

## Возможности
- Изменение радиуса обнаружения.
- Полное отключение обнаружения.
- Изменение цвета точки.

---

## Команды
`/setrange <число>` - Изменяет радиус обнаружения. При вводе числа `0` полностью отключает возможность обнаружения.

`/setcolor <цвет>` - Изменяет цвет вашей точки на локатор-баре.
